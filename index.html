Jiří Lebl's Github Homepage

This here website is where I put some mirrors of some of my book pages, just in case something happens to the main site. All these books are open source (freely available). Here's what's here so far:

  • Notes on Diffy Qs: Differential Equations for Engineers
    Undergraduate differential equations textbook.
  • Guide to Cultivating Complex Analysis: Working the Complex Field
    A graduate one variable complex analysis course for incoming graduate students. This kind of a course is the prerequisite for this several variable complex analysis course.
  • Tasty Bits of Several Complex Variables
    A one-semester course on several complex variables. A good continuation of the one-variable course above.

The book that's not currently here yet is:
Basic Analysis: Introduction to Real Analysis
A two-semester undergraduate course on basic real analysis.
