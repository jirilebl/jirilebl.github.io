Proposition 8.1.14.

Let \(X\) be a vector space and \(d\) a nonnegative integer.
🔗
  1. If \(X\) is spanned by \(d\) vectors, then \(\dim \, X \leq d\text{.}\)
    🔗
    🔗
  2. If \(T\) is a linearly independent set and \(v \in X \setminus \spn (T)\text{,}\) then \(T \cup \{ v \}\) is linearly independent.
    🔗
    🔗
  3. \(\dim \, X = d\) if and only if \(X\) has a basis of \(d\) vectors. In particular, \(\dim \, \R^n = n\text{.}\)
    🔗
    🔗
  4. If \(Y \subset X\) is a vector subspace and \(\dim \, X = d\text{,}\) then \(\dim \, Y \leq d\text{.}\)
    🔗
    🔗
  5. If \(\dim \, X = d\) and a set \(T\) of \(d\) vectors spans \(X\text{,}\) then \(T\) is linearly independent.
    🔗
    🔗
  6. If \(\dim \, X = d\) and a set \(T\) of \(m\) vectors is linearly independent, then there is a set \(S\) of \(d-m\) vectors such that \(T \cup S\) is a basis of \(X\text{.}\)
    🔗
    🔗
