Proposition 3.2.5.

Let \(f \colon S \to \R\) and \(g \colon S \to \R\) be functions continuous at \(c \in S\text{.}\)
🔗
  1. The function \(h \colon S \to \R\) defined by \(h(x) \coloneqq f(x)+g(x)\) is continuous at \(c\text{.}\)
    🔗
    🔗
  2. The function \(h \colon S \to \R\) defined by \(h(x) \coloneqq f(x)-g(x)\) is continuous at \(c\text{.}\)
    🔗
    🔗
  3. The function \(h \colon S \to \R\) defined by \(h(x) \coloneqq f(x)g(x)\) is continuous at \(c\text{.}\)
    🔗
    🔗
  4. If \(g(x) \neq 0\) for all \(x \in S\text{,}\) the function \(h \colon S \to \R\) given by \(h(x) \coloneqq \frac{f(x)}{g(x)}\) is continuous at \(c\text{.}\)
    🔗
    🔗
