Proposition 3.2.2.

Consider a function \(f \colon S \to \R\) defined on a set \(S \subset \R\) and let \(c \in S\text{.}\) Then:
🔗
  1. If \(c\) is not a cluster point of \(S\text{,}\) then \(f\) is continuous at \(c\text{.}\)
    🔗
    🔗
  2. If \(c\) is a cluster point of \(S\text{,}\) then \(f\) is continuous at \(c\) if and only if the limit of \(f(x)\) as \(x \to c\) exists and
    \begin{equation} \lim_{x\to c} f(x) = f(c) . \end{equation}
    🔗
    🔗
  3. The function \(f\) is continuous at \(c\) if and only if for every sequence \(\{ x_n \}_{n=1}^\infty\) where \(x_n \in S\) and \(\lim\limits_{n\to\infty} x_n = c\text{,}\) the sequence \(\bigl\{ f(x_n) \bigr\}_{n=1}^\infty\) converges to \(f(c)\text{.}\)
    🔗
    🔗
