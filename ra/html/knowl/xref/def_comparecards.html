Definition 0.3.28.

We write
\begin{equation} \sabs{A} \leq \sabs{B} \end{equation}
if there exists an injection from \(A\) to \(B\text{.}\) We write \(\sabs{A} = \sabs{B}\) if \(A\) and \(B\) have the same cardinality. We write \(\sabs{A} < \sabs{B}\) if \(\sabs{A} \leq \sabs{B}\text{,}\) but \(A\) and \(B\) do not have the same cardinality.
