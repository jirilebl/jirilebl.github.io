Example 5.1.4.

Take the Dirichlet function \(f \colon [0,1] \to \R\text{,}\) where \(f(x) \coloneqq 1\) if \(x \in \Q\) and \(f(x) \coloneqq 0\) if \(x \notin \Q\text{.}\) Then
\begin{equation} \underline{\int_0^1} f = 0 \qquad \text{and} \qquad \overline{\int_0^1} f = 1 . \end{equation}
The reason is that for any partition \(P\) and every \(i\text{,}\) we have \(m_i = \inf \bigl\{ f(x) : x \in [x_{i-1},x_i] \bigr\} = 0\) and \(M_i = \sup \bigl\{ f(x) : x \in [x_{i-1},x_i] \bigr\} = 1\text{.}\) Thus
\begin{equation} L(P,f) = \sum_{i=1}^n 0 \cdot \Delta x_i = 0 , \quad \text{and} \quad U(P,f) = \sum_{i=1}^n 1 \cdot \Delta x_i = \sum_{i=1}^n \Delta x_i = 1 . \end{equation}
