Proposition 1.1.8.

Let \(F\) be an ordered field and \(x,y,z,w \in F\text{.}\) Then
🔗
  1. If \(x > 0\text{,}\) then \(-x < 0\) (and vice versa).
    🔗
    🔗
  2. If \(x > 0\) and \(y < z\text{,}\) then \(xy < xz\text{.}\)
    🔗
    🔗
  3. If \(x < 0\) and \(y < z\text{,}\) then \(xy > xz\text{.}\)
    🔗
    🔗
  4. If \(x \neq 0\text{,}\) then \(x^2 > 0\text{.}\)
    🔗
    🔗
  5. If \(0 < x < y\text{,}\) then \(0 < \nicefrac{1}{y} < \nicefrac{1}{x}\text{.}\)
    🔗
    🔗
  6. If \(0 < x < y\text{,}\) then \(x^2 < y^2\text{.}\)
    🔗
    🔗
  7. If \(x \leq y\) and \(z \leq w\text{,}\) then \(x + z \leq y + w\text{.}\)
    🔗
    🔗
