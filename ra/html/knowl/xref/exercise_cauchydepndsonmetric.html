Exercise 7.4.19.

Consider two metrics on \(\R\text{.}\) Let \(d(x,y) \coloneqq \sabs{x-y}\) be the standard metric, and let \(d'(x,y) \coloneqq \babs{ \frac{x}{1+\sabs{x}} - \frac{y}{1+\sabs{y}} }\text{.}\)
🔗
  1. Show that \((\R,d')\) is a metric space (if you have done Exercise 7.3.10, the computation is the same).
    🔗
    🔗
  2. Show that the topology is the same, that is, a set is open in \((\R,d)\) if and only if it is open in \((\R,d')\text{.}\)
    🔗
    🔗
  3. Show that a set is compact in \((\R,d)\) if and only if it is compact in \((\R,d')\text{.}\)
    🔗
    🔗
  4. Show that a sequence converges in \((\R,d)\) if and only if it converges in \((\R,d')\text{.}\)
    🔗
    🔗
  5. Find a sequence of real numbers that is Cauchy in \((\R,d')\) but not Cauchy in \((\R,d)\text{.}\)
    🔗
    🔗
  6. While \((\R,d)\) is complete, show that \((\R,d')\) is not complete.
