Proposition 4.2.8.

Let \(I\) be an interval and let \(f \colon I \to \R\) be a differentiable function.
🔗
  1. If \(f'(x) > 0\) for all \(x \in I\text{,}\) then \(f\) is strictly increasing.
    🔗
    🔗
  2. If \(f'(x) < 0\) for all \(x \in I\text{,}\) then \(f\) is strictly decreasing.
    🔗
    🔗
