Proposition 3.1.15.

Let \(S \subset \R\text{,}\) \(c \in \R\text{,}\) and let \(f \colon S \to \R\) be a function. Suppose \(A \subset S\) is such that for some \(\alpha > 0\text{,}\) we have \((A \setminus \{ c \}) \cap (c-\alpha,c+\alpha) = (S \setminus \{ c \}) \cap (c-\alpha,c+\alpha)\text{.}\)
🔗
  1. The point \(c\) is a cluster point of \(A\) if and only if \(c\) is a cluster point of \(S\text{.}\)
    🔗
    🔗
  2. Supposing \(c\) is a cluster point of \(S\text{,}\) then \(f(x) \to L\) as \(x \to c\) if and only if \(f|_A(x) \to L\) as \(x \to c\text{.}\)
