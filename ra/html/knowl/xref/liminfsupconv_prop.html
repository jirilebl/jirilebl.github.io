Proposition 2.3.5.

Let \(\{ x_n \}_{n=1}^\infty\) be a bounded sequence. Then \(\{ x_n \}_{n=1}^\infty\) converges if and only if
\begin{equation} \liminf_{n\to \infty} x_n = \limsup_{n\to \infty} x_n. \end{equation}
Furthermore, if \(\{ x_n \}_{n=1}^\infty\) converges, then
\begin{equation} \lim_{n\to \infty} x_n = \liminf_{n\to \infty} x_n = \limsup_{n\to \infty} x_n. \end{equation}
