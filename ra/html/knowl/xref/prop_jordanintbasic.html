Proposition 10.5.6.

Suppose \(S \subset \R^n\) is a bounded Jordan measurable set and \(f \colon S \to \R\) and \(g \colon S \to \R\) are Riemann integrable on \(S\text{,}\) and \(\alpha \in \R\text{.}\) Then
🔗
  1. If \(f = 0\) almost everywhere, then \(\int_S f = 0\text{.}\)
    🔗
    🔗
  2. If \(f = g\) almost everywhere, then \(\int_S f = \int_S g\text{.}\)
    🔗
    🔗
  3. \(f+g\) is Riemann integrable on \(S\) and \(\int_S (f+g) = \int_S f + \int_S g\text{.}\)
    🔗
    🔗
  4. \(\alpha f\) is Riemann integrable on \(S\) and \(\int_S \alpha f = \alpha \int_S f\text{.}\)
    🔗
    🔗
  5. If \(f(x) \leq g(x)\) for almost every \(x\text{,}\) then \(\int_S f \leq \int_S g\text{.}\)
    🔗
    🔗
