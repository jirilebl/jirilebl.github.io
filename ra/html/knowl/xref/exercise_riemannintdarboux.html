Exercise 5.1.12.

(Challenging)   Generalize the previous exercise. Show that \(f \in \sR\bigl([a,b]\bigr)\) if and only if there exists an \(I \in \R\text{,}\) such that for every \(\epsilon > 0\) there exists a \(\delta > 0\) such that if \(P\) is a partition with \(\Delta x_i < \delta\) for all \(i\text{,}\) then \(\babs{L(P,f) - I} < \epsilon\) and \(\babs{U(P,f) - I} < \epsilon\text{.}\) If \(f \in \sR\bigl([a,b]\bigr)\text{,}\) then \(I = \int_a^b f\text{.}\)
