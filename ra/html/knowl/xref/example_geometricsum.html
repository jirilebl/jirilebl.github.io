Example 0.3.8.

We claim that for all \(c \neq 1\) and all \(n \in \N \cup \{ 0 \}\text{,}\)
\begin{equation} 1 + c + c^2 + \cdots + c^n = \frac{1-c^{n+1}}{1-c} . \end{equation}
🔗
Proof: Fix some \(c \neq 1\text{.}\) It is easy to check that the equation holds with \(n=0\text{.}\) Suppose it holds for some \(n\) (including possibly \(n=0\)). Then
\begin{equation} \begin{split} 1 + c + c^2 + \cdots + c^n + c^{n+1} & = ( 1 + c + c^2 + \cdots + c^n ) + c^{n+1} \\ & = \frac{1-c^{n+1}}{1-c} + c^{n+1} \\ & = \frac{1-c^{n+1} + (1-c)c^{n+1}}{1-c} \\ & = \frac{1-c^{n+2}}{1-c} . \end{split} \end{equation}
By the principle of induction, the equation holds for all \(n \in \N \cup \{ 0 \}\text{.}\)
