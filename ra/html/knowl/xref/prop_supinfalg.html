Proposition 1.2.6.

Let \(A \subset \R\) be nonempty.
🔗
  1. If \(x \in \R\) and \(A\) is bounded above, then \(\sup (x+A) = x + \sup\, A\text{.}\)
    🔗
    🔗
  2. If \(x \in \R\) and \(A\) is bounded below, then \(\inf (x+A) = x + \inf\, A\text{.}\)
    🔗
    🔗
  3. If \(x > 0\) and \(A\) is bounded above, then \(\sup (xA) = x ( \sup\, A )\text{.}\)
    🔗
    🔗
  4. If \(x > 0\) and \(A\) is bounded below, then \(\inf (xA) = x ( \inf\, A )\text{.}\)
    🔗
    🔗
  5. If \(x < 0\) and \(A\) is bounded below, then \(\sup (xA) = x ( \inf\, A )\text{.}\)
    🔗
    🔗
  6. If \(x < 0\) and \(A\) is bounded above, then \(\inf (xA) = x ( \sup\, A )\text{.}\)
    🔗
    🔗
