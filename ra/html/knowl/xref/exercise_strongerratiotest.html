Exercise 2.5.6.

Prove the following stronger version of the ratio test: Let \(\sum_{n=1}^\infty x_n\) be a series.
🔗
  1. If there is an \(N\) and a \(\rho < 1\) such that \(\frac{\sabs{x_{n+1}}}{\sabs{x_n}} < \rho\) for all \(n \geq N\text{,}\) then the series converges absolutely. (Remark: Equivalently the condition can be stated as \(\limsup\limits_{n\to\infty} \frac{\sabs{x_{n+1}}}{\sabs{x_n}} < 1\text{.}\))
    🔗
    🔗
  2. If there is an \(N\) such that \(\frac{\sabs{x_{n+1}}}{\sabs{x_n}} \geq 1\) for all \(n \geq N\text{,}\) then the series diverges.
    🔗
    🔗
