Exercise 2.2.13.

(Easy)   Prove the following stronger version of Lemma 2.2.12, the ratio test. Suppose \(\{ x_n \}_{n=1}^\infty\) is a sequence such that \(x_n \neq 0\) for all \(n\text{.}\)
🔗
  1. Prove that if there exists an \(r < 1\) and \(M \in \N\) such that
    \begin{equation} \frac{\sabs{x_{n+1}}}{\sabs{x_n}} \leq r \quad \text{for all } n \geq M, \end{equation}
    then \(\{ x_n \}_{n=1}^\infty\) converges to \(0\text{.}\)
    🔗
    🔗
  2. Prove that if there exists an \(r > 1\) and \(M \in \N\) such that
    \begin{equation} \frac{\sabs{x_{n+1}}}{\sabs{x_n}} \geq r \quad \text{for all } n \geq M, \end{equation}
    then \(\{ x_n \}_{n=1}^\infty\) is unbounded.
    🔗
    🔗
