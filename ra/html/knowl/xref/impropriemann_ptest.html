Proposition 5.5.2. \(p\)-test for integrals.

The improper integral
\begin{equation} \int_1^\infty \frac{1}{x^p} \,dx \end{equation}
converges to \(\frac{1}{p-1}\) if \(p > 1\) and diverges if \(0 < p \leq 1\text{.}\)
🔗
The improper integral
\begin{equation} \int_0^1 \frac{1}{x^p} \,dx \end{equation}
converges to \(\frac{1}{1-p}\) if \(0 < p < 1\) and diverges if \(p \geq 1\text{.}\)
