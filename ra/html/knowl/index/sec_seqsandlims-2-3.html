Definition 2.1.1.

A sequence (of real numbers) is a function \(x \colon \N \to \R\text{.}\) Instead of \(x(n)\text{,}\) we usually denote the \(n\)th element in the sequence by \(x_n\text{.}\) To denote a sequence we write
 1 
It is common to use \(\{ x_n \}\) or \(\{ x_n \}_n\) for brevity.
\begin{equation} \{ x_n \}_{n=1}^\infty. \end{equation}
🔗
A sequence \(\{ x_n \}_{n=1}^\infty\) is bounded if the underlying function is bounded. That is, if there exists a \(B \in \R\) such that
\begin{equation} \sabs{x_n} \leq B \qquad \text{for all } n \in \N. \end{equation}
In other words, the sequence \(\{x_n\}_{n=1}^\infty\) is bounded whenever the set \(\{ x_n : n \in \N \}\) is bounded. We similarly define the words bounded below and bounded above.
