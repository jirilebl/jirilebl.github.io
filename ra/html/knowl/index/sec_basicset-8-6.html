Definition 0.3.27.

If \(A\) has the same cardinality as \(\{ 1,2,3,\ldots,n \}\) for some \(n \in \N\text{,}\) we write \(\sabs{A} \coloneqq n\text{.}\) If \(A\) is empty, we write \(\sabs{A} \coloneqq 0\text{.}\) In either case, we say that \(A\) is finite. We say \(A\) is infinite or “of infinite cardinality” if \(A\) is not finite.
