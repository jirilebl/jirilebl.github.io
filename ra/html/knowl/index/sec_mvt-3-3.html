Definition 4.2.1.

Let \(S \subset \R\) be a set and let \(f \colon S \to \R\) be a function. The function \(f\) is said to have a relative maximum at \(c \in S\) if there exists a \(\delta>0\) such that for all \(x \in S\) where \(\sabs{x-c} < \delta\text{,}\) we have \(f(x) \leq f(c)\text{.}\) The definition of relative minimum is analogous.
