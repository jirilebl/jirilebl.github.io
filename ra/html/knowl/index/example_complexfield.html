Example 1.1.10.

The reader may have heard about the complex numbers, usually denoted by \(\C\text{.}\) That is, \(\C\) is the set of numbers of the form \(x + iy\text{,}\) where \(x\) and \(y\) are real numbers, and \(i\) is the imaginary number, a number such that \(i^2 = -1\text{.}\) The reader may remember from algebra that \(\C\) is also a field; however, it is not an ordered field. While one can make \(\C\) into an ordered set in some way, it is not possible to put an order on \(\C\) that would make it an ordered field: In every ordered field, \(-1 < 0\) and \(x^2 > 0\) for all nonzero \(x\text{,}\) but in \(\C\text{,}\) \(i^2 = -1\text{.}\)
