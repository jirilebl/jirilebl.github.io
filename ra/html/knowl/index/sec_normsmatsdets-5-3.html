Define the symbol \(\operatorname{sgn}(x)\) (read “sign of \(x\)”) for a number \(x\) by
\begin{equation} \operatorname{sgn}(x) \coloneqq \begin{cases} -1 & \text{if } x < 0 , \\ 0 & \text{if } x = 0 , \\ 1 & \text{if } x > 0 . \end{cases} \end{equation}
A permutation \(\sigma = (\sigma_1,\sigma_2,\ldots,\sigma_n)\) is a reordering of \((1,2,\ldots,n)\text{.}\) Define
\begin{equation} \operatorname{sgn}(\sigma) = \operatorname{sgn}(\sigma_1,\ldots,\sigma_n) \coloneqq \prod_{p < q} \operatorname{sgn}(\sigma_q-\sigma_p) .\tag{8.4} \end{equation}
Here \(\prod\) stands for multiplication, similarly to how \(\sum\) stands for summation.
