Definition 2.1.9.

A sequence \(\{ x_n \}_{n=1}^\infty\) is monotone increasing if \(x_n \leq x_{n+1}\) for all \(n \in \N\text{.}\) A sequence \(\{ x_n \}_{n=1}^\infty\) is monotone decreasing if \(x_n \geq x_{n+1}\) for all \(n \in \N\text{.}\) If a sequence is either monotone increasing or monotone decreasing, we can simply say the sequence is monotone.
