A typical application of this proposition (together with Proposition 3.1.17) is the following. If \(f \colon (-1,0) \cup (0,1) \to \R\) is uniformly continuous, then \(\lim_{x\to 0} f(x)\) exists and the function has a removable singularity. That is, we can extend the function to a continuous function on \((-1,1)\text{.}\)
