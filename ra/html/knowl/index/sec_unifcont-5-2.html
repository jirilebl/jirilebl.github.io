Definition 3.4.7.

A function \(f \colon S \to \R\) is Lipschitz continuous
 1 
Named after the German mathematician Rudolf Otto Sigismund Lipschitz (1832–1903).
if there exists a \(K \in \R\) such that
\begin{equation} \babs{f(x)-f(y)} \leq K \sabs{x-y} \qquad \text{for all } x \text{ and } y \text{ in } S. \end{equation}
