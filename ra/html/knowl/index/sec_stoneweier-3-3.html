Theorem 11.7.1. Weierstrass approximation theorem.

If \(f \colon [a,b] \to \C\) is continuous, then there exists a sequence \(\{ p_n \}_{n=1}^\infty\) of polynomials converging to \(f\) uniformly on \([a,b]\text{.}\) Furthermore, if \(f\) is real-valued, we can find \(p_n\) with real coefficients.
