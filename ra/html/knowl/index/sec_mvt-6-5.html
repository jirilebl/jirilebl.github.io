Now that we know what it means for the function to stay constant, we look at increasing and decreasing functions. We say \(f \colon I \to \R\) is increasing (resp. strictly increasing) if \(x < y\) implies \(f(x) \leq f(y)\) (resp. \(f(x) < f(y)\)). We define decreasing and strictly decreasing in the same way by switching the inequalities for \(f\text{.}\)
