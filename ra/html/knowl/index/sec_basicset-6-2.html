Informally, a set-theoretic function \(f\) taking a set \(A\) to a set \(B\) is a mapping that to each \(x \in A\) assigns a unique \(y \in B\text{.}\) We write \(f \colon A \to B\text{.}\) A function \(f \colon A \to B\) is a black box into which we stick an element of \(A\text{,}\) and the function spits out an element of \(B\text{.}\) Sometimes \(f\) is called a mapping or a map, and we say \(f\) maps \(A\) to \(B\).
