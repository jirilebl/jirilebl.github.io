Proposition 8.2.6.

Let \(X\) be a finite-dimensional normed vector space. Let \(GL(X) \subset L(X)\) be the set of invertible linear operators.
 2 
\(GL(X)\) is called the general linear group, that is where the acronym GL comes from.
🔗
  1. If \(A \in GL(X)\text{,}\) \(B \in L(X)\text{,}\) and
    \begin{equation} \snorm{A-B} < \frac{1}{\snorm{A^{-1}}},\tag{8.2} \end{equation}
    then \(B \in GL(X)\text{,}\) that is, \(B\) is invertible. In particular, \(GL(X)\) is open.
    🔗
    🔗
  2. \(A \mapsto A^{-1}\) is a continuous function on \(GL(X)\text{.}\)
