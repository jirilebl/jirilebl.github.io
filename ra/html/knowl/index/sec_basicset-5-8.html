The assumption that \(P(n)\) is true in “if \(P(n)\) is true, then \(P(n+1)\) is true” is usually called the induction hypothesis. Sometimes it is convenient to start induction at a different number than 1, all that changes is the labeling. For example, if starting say at \(n=0\text{,}\) then \(P(0)\) is the basis statement, and the induction step must be proved for all \(n \in \N \cup \{ 0 \}\text{.}\) The conclusion is then that \(P(n)\) is true for all \(n \in \N \cup \{ 0 \}\text{.}\)
