Proposition 2.5.16. Comparison test.

Let \(\sum_{n=1}^\infty x_n\) and \(\sum_{n=1}^\infty y_n\) be series such that \(0 \leq x_n \leq y_n\) for all \(n \in \N\text{.}\)
🔗
  1. If \(\sum_{n=1}^\infty y_n\) converges, then so does \(\sum_{n=1}^\infty x_n\text{.}\)
    🔗
    🔗
  2. If \(\sum_{n=1}^\infty x_n\) diverges, then so does \(\sum_{n=1}^\infty y_n\text{.}\)

    🔗
