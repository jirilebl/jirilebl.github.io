Definition 7.3.1.

A sequence in a metric space \((X,d)\) is a function \(x \colon \N \to X\text{.}\) As before we write \(x_n\) for the \(n\)th element in the sequence, and for the whole sequence use the notation
\begin{equation} \{ x_n \}_{n=1}^\infty . \end{equation}
🔗
A sequence \(\{ x_n \}_{n=1}^\infty\) is bounded if there exists a point \(p \in X\) and \(B \in \R\) such that
\begin{equation} d(p,x_n) \leq B \qquad \text{for all } n \in \N. \end{equation}
In other words, the sequence \(\{x_n\}_{n=1}^\infty\) is bounded whenever the set \(\{ x_n : n \in \N \}\) is bounded.
🔗
If \(\{ n_k \}_{k=1}^\infty\) is a sequence of natural numbers such that \(n_{k+1} > n_k\) for all \(k\text{,}\) then the sequence \(\{ x_{n_k} \}_{k=1}^\infty\) is said to be a subsequence of \(\{x_n \}_{n=1}^\infty\text{.}\)
