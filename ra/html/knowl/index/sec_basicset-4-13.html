To define new sets, one often uses the set building notation,
\begin{equation} \bigl\{ x \in A : P(x) \bigr\} . \end{equation}
This notation refers to a subset of the set \(A\) containing all elements of \(A\) that satisfy the property \(P(x)\text{.}\) Using \(S = \{ 0, 1, 2 \}\) as above, \(\{ x \in S : x \neq 2 \}\) is the set \(\{ 0, 1 \}\text{.}\) The notation is sometimes abbreviated as \(\bigl\{ x : P(x) \bigr\}\text{,}\) that is, \(A\) is not mentioned when understood from context. Furthermore, \(x \in A\) is sometimes replaced with a formula to make the notation easier to read.
