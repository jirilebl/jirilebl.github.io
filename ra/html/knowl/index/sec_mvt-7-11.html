It is sometimes useful to assume the derivative of a differentiable function is continuous. If \(f \colon I \to \R\) is differentiable and the derivative \(f'\) is continuous on \(I\text{,}\) then we say \(f\) is continuously differentiable. It is common to write \(C^1(I)\) for the set of continuously differentiable functions defined on \(I\text{.}\)
