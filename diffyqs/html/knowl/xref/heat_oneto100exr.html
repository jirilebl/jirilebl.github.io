Exercise 4.6.6.

Find a series solution of
\begin{equation} \begin{array}{ll} u_t = u_{xx} & \qquad \text{for } \; 0 < x < 1 \text{ and } t > 0, \\ u(0,t) = 0 , \quad u(1,t) = 100 & \qquad \text{for } \; t > 0, \\ u(x,0) = \sin (\pi x) & \qquad \text{for } \; 0 < x < 1 . \end{array} \end{equation}
Hint: Use the fact that \(u(x,t) = 100 x\) is a solution satisfying \(u_t = u_{xx}\text{,}\) \(u(0,t) = 0\text{,}\) \(u(1,t) = 100\text{.}\) Then use superposition.
