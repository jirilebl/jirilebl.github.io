In real applications, we would not use a simple method such as Euler’s. The simplest method one would probably use in a real application is the standard Runge–Kutta method (see exercises). That is a fourth-order method, meaning that if we halve the step size, the error gets multiplied by \(\nicefrac{1}{16}\) (it is fourth-order as \(\nicefrac{1}{16} = \nicefrac{1}{2} \times \nicefrac{1}{2} \times \nicefrac{1}{2} \times \nicefrac{1}{2}\)).
