Hence \(e^{tP}\) is a fundamental matrix solution of the homogeneous system. If we can compute the matrix exponential, we have another method of solving constant-coefficient homogeneous systems. It also makes it easy to solve for initial conditions. To solve \({\vec{x}}' = P \vec{x}\text{,}\) \(\vec{x}(0) = \vec{b}\text{,}\) we take the solution
\begin{equation} \vec{x} = e^{tP} \vec{b} . \end{equation}
This equation follows because \(e^{0P} = I\text{,}\) so \(\vec{x} (0) = e^{0P} \vec{b} = \vec{b}\text{.}\)
