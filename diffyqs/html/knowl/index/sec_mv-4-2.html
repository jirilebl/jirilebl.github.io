In this section we only consider free or unforced motion, as we do not know yet how to solve nonhomogeneous equations. We start with undamped motion where \(c=0\text{.}\) The equation is
\begin{equation} mx'' + kx = 0 . \end{equation}
We divide by \(m\) and let \(\omega_0 = \sqrt{\nicefrac{k}{m}}\) to rewrite the equation as
\begin{equation} x'' + \omega_0^2 x = 0 . \end{equation}
The general solution to this equation is
\begin{equation} x(t) = A \cos (\omega_0 t) + B \sin (\omega_0 t) . \end{equation}
By a trigonometric identity
\begin{equation} A \cos (\omega_0 t) + B \sin (\omega_0 t) = C \cos ( \omega_0 t - \gamma ) , \end{equation}
for two different constants \(C\) and \(\gamma\text{.}\) One finds that \(A = C \cos \gamma\) and \(B = C \sin \gamma\text{,}\) and therefore it is not hard to compute that \(C= \sqrt{A^2 + B^2}\) and \(\tan \gamma = \nicefrac{B}{A}\text{.}\) Therefore, we let \(C\) and \(\gamma\) be our arbitrary constants and write \(x(t) = C \cos ( \omega_0 t - \gamma )\text{.}\)
