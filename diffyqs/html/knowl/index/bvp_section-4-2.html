For basic Fourier series theory, we will need the following three eigenvalue problems:
\begin{equation} x'' + \lambda x = 0, \quad x(a) = 0, \quad x(b) = 0 ,\tag{4.1} \end{equation}
\begin{equation} x'' + \lambda x = 0, \quad x'(a) = 0, \quad x'(b) = 0 ,\tag{4.2} \end{equation}
and
\begin{equation} x'' + \lambda x = 0, \quad x(a) = x(b), \quad x'(a) = x'(b) .\tag{4.3} \end{equation}
A number \(\lambda\) is called an eigenvalue of (4.1) (resp. (4.2) or (4.3)) if and only if there exists a nonzero (not identically zero) solution to (4.1) (resp. (4.2) or (4.3)) given that specific \(\lambda\text{.}\) A nonzero solution is called a corresponding eigenfunction.
